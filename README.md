# react_questiona
给刚开始接触前端 react 框架的人练手的小项目

## 如何使用

下载

```bash
git clone git@github.com:lawler61/react_questiona.git
```

安装

```bash
npm install
```

启动

```bash
npm start
or
yarn start
```
