# react_questiona
给刚开始接触前端 react 框架的人练手的小项目。

使用`create-react-app`初始化项目

版本信息如下：

`node: v8.11.2`

`npm: 5.6.0`

`yarn: v1.7.0`

## 如何使用

下载完后，进入项目文件

```bash
D:\>cd D:\文件存放的位置\react_questiona\react_questiona
```

然后npm安装

```bash
npm install
```

最后yarn启动

```bash
yarn start
```

通过localhost访问

```bash
http://localhost:3000
```
